- Evaluation metrics: **RMSE**, **MAE**, and **R² Score**

---

# 🌧️ Real-Time Sentiment Analysis System

## 📝 Project Description

A **Python-based Real-Time Sentiment Analysis System** that automatically fetches Amazon product reviews using **Rainforest API** and performs comprehensive AI-powered sentiment analysis using Natural Language Processing (NLP) techniques.

---

## 🎯 What Does It Do?

This application collects real Amazon customer reviews and analyzes their sentiment (Positive, Negative, or Neutral) using machine learning algorithms. It provides detailed insights into customer opinions, helping businesses understand product reception and customer satisfaction.

---

## ⚡ Key Features

- **🔄 Real-Time Data Collection** - Fetches live reviews from Amazon
- **🤖 AI Sentiment Detection** - Analyzes emotions using TextBlob NLP
- **📊 Professional Visualizations** - Generates 9 detailed graphs
- **✅ Verified Purchase Analysis** - Tracks authentic customer reviews
- **📈 Rating Correlation** - Compares Amazon ratings vs AI sentiment
- **☁️ Word Cloud Generation** - Visual representation of key terms
- **📉 Statistical Metrics** - Polarity, subjectivity, and variance analysis

---

## 🛠️ Technologies Used

- **Language**: Python 3.8+
- **API**: Rainforest API (Amazon data)
- **NLP Library**: TextBlob
- **Data Processing**: Pandas, NumPy
- **Visualization**: Matplotlib, Seaborn, WordCloud
- **Web Scraping**: Requests, BeautifulSoup4

---

## 🚀 How It Works

### 10-Step Analysis Pipeline:

1. **Data Collection** - Fetch reviews via Rainforest API
2. **Preprocessing** - Clean and normalize text data
3. **Tokenization** - Break text into words
4. **Feature Extraction** - Calculate polarity & subjectivity
5. **Classification** - Categorize as Positive/Negative/Neutral
6. **Scoring** - Convert to 0-100 scale
7. **Metrics Calculation** - Statistical analysis
8. **Advanced Analysis** - Correlation & patterns
9. **Visualization** - Generate 9 professional graphs
10. **Summary Report** - Final insights and conclusions

---

## 📊 Output

- **Console Report**: Detailed step-by-step analysis
- **Visual Graphs**: 9 high-resolution charts (300 DPI)
- **Metrics**: Accuracy, correlation, distribution statistics
- **Insights**: Key findings and product sentiment summary
